## How to setup locally ?

1. Fork the repo.

2. Clone the repo using the forked repo.

3. Open the code in your favorite code editor.

4. Install the dependencies using the following command:

```
$ flutter pub get
```

5. Build the app using the following command:

```
$ flutter run
```
